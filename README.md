# consulta-frete-api
API REST de consulta de endereço e cálculo de frete para um determinado CEP.

- Java 11
- Spring boot
- API REST Template
- Documentação Swagger
- Testes unitários JUnit5
- Testes automatizados utilizando cucumber(WIP)
- Elastic Beanstalk e S3(AWS)
